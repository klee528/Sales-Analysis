# 🌐 Overview

This project contains an interactive Power BI dashboard created to analyze and visualize the sales data of a pizza store in 2015.

The dashboard provides valuable insights into the overall business performance of the store through dynamic visualizations and key performance indicatior. It helps users better understand sales trends, customer purchasing behavior and product performance.

# 🔖 Tools & Technologies 

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
- CSV Dataset

# 📁 Project Files

- 📘 **`Pizza sales.pbix`**
- 💰 **`pizza_sales.csv`**
- 📑 **`README.md`** 

# 🔍 Key Insights

- Identified the busiest sales periods of the year
- Analyzed revenue trends over time
- Determined the most popular pizz categories and sizes
- Evaluated overall business performance
